Commands

/antilink on - If any messages contains a link. Selena will delete it. Goes into antilink mode.

/antilink off - Selena goes out of antilink mode.

/admins - Replies with a list of all group administrators

/ban - Removes the quoted member from the group

/ban [phone_number / mention the member] [Reason for ban] - Removes the member from the group and replies with the reason

/chat - Selena will prompt GPT by AI/ML Api and return a response to the message sent. This is a third party service... Learn more

/del - Deletes the quoted message. You can also use /delete

/demote [phone_number / mention the member] - Remove the admin rights from an admin

/exit - Selina will leave the group

/join [invite_link] - Selina will join the group with the invite link provided

/link - Replies with the group's invite link

/pin [number] [timeframe (seconds/minutes/hours/days)] - Pins the message for the specified timeframe. e.g. "/pin 4 days" pins the message for 4 days

/promote [phone_number / mention the member] - Make the mentioned member an admin

/status - Checks whether the bot is online

/tagadmins - Selina will tag all administrators of the group

/tagall - Selina will tag all members of the group

/warn - Selena sends a warning message to the member. Three warns will automatically lead to a removal from the group.

/add - This feature isn't available publicly due to Whatsapp's Privacy Policy. If you need it, contact me... Telegram
